'''
Hello this project aims on building a simple application
using django python framework and simple HTML a Blog post 
website and use the default sqlite as a database connection.
'''


'''
Start the project 
'''
python manage.py runserver  
'''
This will run up the server  
Make sure you have made the necessary migrations for the project  
I had created a virtualenv using python before building this app 
so if you are having issues with the project run it by creating 
your own virtualenv named blogpost and then try it 


To make migrations   

'''
python manage.py makemigrations 
'''



To migrate  
    
'''
python manage.py migrate
'''
